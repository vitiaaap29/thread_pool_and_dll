Вторая и третья лабораторная по ОСиСПу.
Создатели:
	1. Пунько Виктор - пул потоков, копирование, размер
	2. Максим Ледян - динамические и статические dll.
	3. Алексей Макарычев - идеи, тестирование и моральная поддержка=)
Параметры командной строки: 
	Минимальное число потоков в пуле, максимальное число потоков в пуле, максимальное время
	жизни бездействующего потока, после которого он завершается специальным потоком-убийцей.
При запуске из Visual Studio по умолчанию ставятся параметры: 2 10 1000.

Это приложение OSiSP создано автоматически с помощью мастера приложений.

В этом файле представлена сводка содержимого всех файлов, входящих в состав приложения OSiSP.


OSiSP.vcxproj
    Это основной файл проекта VC++, создаваемый с помощью мастера приложений. Он содержит данные о версии языка Visual C++, использованной для создания файла, а также сведения о платформах, конфигурациях и функциях проекта, выбранных с помощью мастера приложений.

OSiSP.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера приложений. Он содержит сведения о сопоставлениях между файлами в вашем проекте и фильтрами. Эти сопоставления используются в среде IDE для группировки файлов с одинаковыми расширениями в одном узле (например CPP-файлы сопоставляются с фильтром "Исходные файлы").

OSiSP.cpp
    Это основной исходный файл приложения.

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
    Эти файлы используются для построения файла предкомпилированного заголовка (PCH) с именем OSiSP.pch и файла предкомпилированных типов с именем StdAfx.obj.
